# Changes September 2026

## IT changes

From the academic year 2025-2026 to 2026-2027 the following changes have been made:

- Assignments in the workbook can now be downloaded as a zip
- The FTP-server is no longer used for storing big files, instead we use Git LFS storage (see [this page](./LFS.md) for more information)
- A GitHub app is used to automatically merge changes to published versions and assignment and solution versions instead of a token on Tom's account.
- Py files are now also supported for the assignment and solution version generation (see [this page](./assignment_repo.md) for more information)
- Assignment repos now share the same git history so that formatting changes can be made across all repos at once.
- Assignment files no longer use HTML blocks
- The book and workbook now allows for scheduling merging of pull requests.

## Content changes

- Week 1 focusses on git.
- Zheng Guan is now in charge of univariate distributions
- An appendix is added to the textbook for prerequisite material. This replaces the fundamental concepts book.
- Week 2.7 and 2.8 are combining their group assignments to a bigger project.

## Organizational changes

- Question hours in Q2 are no longer held.
- Assignments are no longer provided via GitHub classroom but as template repositories.
